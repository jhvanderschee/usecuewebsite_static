Instagram interface iconen

Ik heb een webtool ontwikkelt waarmee je Instagram eenvoudig op je website kunt publiceren, genaamd ‘PPI Loader’.

Om te illustreren hoe deze tool werkt, heb ik een demo gemaakt. Voor deze demo gebruikte ik de originele interface iconen van Instagram, maar deze waren tot mijn verbazing in PNG formaat en niet in SVG (vector). Online kon ik gek genoeg ook geen vectorvarianten van deze iconen vinden. Daarom heb ik wat van mijn tijd opgeofferd om ze voor jullie te maken.
